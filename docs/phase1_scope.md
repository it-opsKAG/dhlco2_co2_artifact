# Phase 1 Scope

This repository is the SSOT for Phase 1 outputs.

- Vault receives promoted summaries.
- Jira receives mirrored progress notes.
- Unknowns are encoded as `TBD` and tracked in `exports/Gap_Report.md`.
- Promotion state is tracked in `system/registry/staging_promotions.v1.yaml`.
- Staging archive moves are tracked in `system/registry/staging_archives.v1.yaml`.
