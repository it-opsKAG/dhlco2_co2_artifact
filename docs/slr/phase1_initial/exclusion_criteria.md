# Exclusion Criteria

1. Sources without explicit method description for quantification or boundary setting.
2. Sources focused only on generic sustainability claims without calculable metrics.
3. Sources that cannot be traced to a verifiable publication, standard, or institutional source.
4. Sources not relevant to software, digital operations, or cloud/CI workloads.
5. Sources that duplicate another source with lower methodological quality.
6. Sources that require proprietary assumptions that cannot be documented in the evidence ledger.
